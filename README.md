# 🌤️ Weather App

A simple weather web application that allows users to search any city and view real-time temperature and weather conditions.

Built using HTML, CSS, and JavaScript, and powered by WeatherAPI.

---

## 🚀 Features

- Search weather by city name  
- Displays current temperature (°C)  
- Shows weather condition (e.g. Sunny, Cloudy)  
- Clean and modern UI (Black, Gold, White theme)  
- Fast and responsive  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript (Fetch API)  
- WeatherAPI  

## ⚙️ How It Works

1. User enters a city name  
2. App sends request to WeatherAPI  
3. API returns real-time weather data  
4. Data is displayed on the screen
